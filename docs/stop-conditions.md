# Stop Conditions

Stop implementation and keep the PR Draft when any of these conditions is true:

- input or source rights are unclear;
- a style conflict has no owner decision;
- the validator fails;
- validator execution cannot be reproduced on the exact HEAD;
- a critical visual defect remains;
- the final image is not visible to the user;
- a requested runtime capability is unavailable and no documented fallback exists;
- the owner has not issued a separate Ready or merge instruction.
