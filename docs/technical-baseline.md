# Technical Baseline

The initial baseline is considered structurally complete when the Draft PR contains:

- plugin manifest;
- one canonical orchestration skill;
- canonical reference documents;
- three versioned style packs;
- Scene Specification and output manifest schemas;
- deterministic validator;
- positive and negative fixtures;
- source traceability and ADRs;
- asset manifest and checksums;
- explicit validation and lifecycle evidence policy.

This baseline is not equivalent to a production release.
