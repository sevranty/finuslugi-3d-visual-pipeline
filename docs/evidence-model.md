# Evidence Model

## Authoritative facts

- Issue defines intent and Definition of Done.
- Branch and commits define implementation history.
- Draft PR defines the review boundary.
- Validation report defines deterministic check results on an exact HEAD.
- Review comment defines owner-review findings on an exact HEAD.
- Asset manifest and checksums define binary provenance.
- Eval manifests define behavioral and visual evidence.

No single document duplicates all facts. Each artifact links to the authoritative source for its layer.
