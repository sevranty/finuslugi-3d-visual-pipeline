- Partial #4 — behavior cases and schema fixtures, without golden visual results.

## Excluded Issues

- #6 runtime adapter implementation.

## Merge policy

The PR remains Draft. It must not be marked Ready or merged until:

1. the validator runs against the exact PR HEAD;
2. findings are resolved;
3. owner review is repeated on the unchanged validated HEAD;
4. the owner gives a separate lifecycle instruction.
