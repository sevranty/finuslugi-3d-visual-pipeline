# Generation Sequence

## Simple scene

```text
P1 base generation
  -> QA meaning and composition
P2 local form or scale correction
  -> QA material and light
P3 final integration
  -> detail and upscale
```

## Complex scene

```text
G1 composition + camera + scale
  -> approve composition
G2 form + construction + proportions
  -> approve geometry
G3 materials
  -> approve material system
G4 lighting + shadows
  -> approve light logic
G5 reflections
  -> approve integration
G6 final detail
  -> final QA
G7 upscale and export
```

Do not move to a later layer while an earlier critical gate is failing.

## First-generation purpose

The first generation is a structural draft. Evaluate only:

- whether the metaphor reads;
- whether the main object is correct;
- whether scale and position are correct;
- whether the source form has been preserved;
- whether the chosen style family is recognizable.

Do not reject the first draft for final micro-detail that belongs to a later stage.
