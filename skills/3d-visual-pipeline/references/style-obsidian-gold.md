# Obsidian Gold 1.0

Use one isolated object on `#000000`. The object is matte obsidian stone with low-poly, faceted, or parametric geometry. Gold is limited to 15-25%. Use soft frontal light, no HDR, and no mirror reflections. Prompt terminator: `The background is black.`
