# Modern Flat 2.1

Use geometric forms, controlled volumetric gradients, directional backlight, clean shadows, and low visual noise. Photorealism is prohibited. `accent_color` is configurable; default `#6B7280`. Accent dominance must not exceed 10%. Maintain 15% vertical and 10% horizontal safe zones.
